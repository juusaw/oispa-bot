# Oispa-bot

A quick Twitter bot to aggregate tweets beginning with a certain phrase.

Tweets with handle [@oispa](https://twitter.com/oispabot).

## Built with

- Plain old Javascript (Node.js)
- AWS Lambda serverless functions
- DynamoDB
